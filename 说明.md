# Tarot Mirror - 塔罗轻治愈应用

## 📖 项目简介

**Tarot Mirror** 是一款面向全球市场的塔罗+轻治愈应用，通过塔罗占卜进行自我探索与心理疗愈，提供情绪支持、行动建议和长期陪伴。

### 核心价值
- **1-Minute Ritual**：每天60-90秒完成"抽牌 + 微任务 + 日历印章"
- **温柔陪伴**：更温柔、更像被陪伴的体验
- **完整闭环**：从占卜到行动的完整闭环

## 🎯 目标市场

- **主推市场**：英文区（欧美市场）
- **次要市场**：中文区
- **目标用户**：18-50岁，spiritual-curious / self-growth / 轻疗愈人群

## 🛠️ 技术方案

- **前端**：Flutter（跨平台）
- **设计**：Figma
- **后端**：待定
- **AI服务**：待定（多LLM支持）
