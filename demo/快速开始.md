# 快速启动指南

## 🚀 最快启动方式

### 步骤 1：进入 demo 目录

```bash
cd /Users/lit/Desktop/tarot/demo
```

### 步骤 2：启动本地服务器

#### 选项 A：使用 Python（最简单）

```bash
python3 -m http.server 8000
```

#### 选项 B：使用 Node.js

```bash
# 如果没有安装 http-server，先安装
npm install -g http-server

# 启动服务器
http-server -p 8000
```

### 步骤 3：打开浏览器

访问：**http://localhost:8000**

## ⚠️ 常见问题

### 问题 1：API 调用失败

**原因**：CORS 错误或 API Key 无效

**解决**：
- 确保通过 HTTP 服务器访问（不是直接打开文件）
- 检查 API Key 是否正确

### 问题 2：图片不显示

**原因**：图片路径错误

**解决**：
- 确保 `Cards-png` 文件夹在 `demo` 目录下
- 检查图片文件名是否正确

### 问题 3：数据没有保存

**原因**：浏览器阻止了 localStorage

**解决**：
- 检查浏览器是否允许本地存储
- 清除浏览器缓存后重试
